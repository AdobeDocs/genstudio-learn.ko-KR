# 마케팅 민첩성: 크리에이티브 이외의 사용자가 확장할 수 있도록 권한 부여

Adobe [!DNL GenStudio], 경우에 따라 완전한 크리에이티브 팀/스튜디오 처리 없이 콘텐츠 요청을 처리할 수 있습니다. Adobe은 마케터와 다른 잠재적인 콘텐츠 작성자의 손에 도구를 넣습니다. 이러한 도구를 사용하면 콘텐츠 변형을 만들 때 빠르게 켤 수 있습니다. 업데이트는 콘텐츠 공급망에 연결된 광범위한 개인 사용자 그룹에서 실행할 수 있으므로 콘텐츠 처리량과 마켓 출시 시간이 크게 늘어납니다.

마케팅 민첩성에 사용되는 Adobe 솔루션은 다음과 같습니다.

## 주요 비즈니스 사용 사례

* 신속한 응답 캠페인을 위한 컨텐츠 생성
* 브랜드를 유지하면서 기존 콘텐츠 리믹스
* 파생 자산(크기, 채널) 생산 자동화
